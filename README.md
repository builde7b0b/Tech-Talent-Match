# Tech Talent Match

A desktop application built with Rust and Slint for matching tech talent based on skills.

## Overview

Tech Talent Match is an open-source desktop application that helps connect developers with opportunities based on their skill sets. Built using Rust and the Slint UI framework, it provides a simple and efficient interface for talent registration and matching.

## Features

- User registration interface
- Skills input and parsing
- Clean and minimal UI design
- Cross-platform compatibility

## Getting Started

### Prerequisites

- Rust (latest stable version)
- Cargo package manager

### Installation

1. Clone the repository:

```bash
git clone https://github.com/yourusername/tech_talent_match.git
cd tech_talent_match

Build and run the application:
cargo run
```
